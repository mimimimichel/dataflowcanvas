# **App Name**: DataFlow Canvas

## Core Features:

- Interactive Data Pipeline Canvas: Interactive canvas for data pipeline visualization with drag-and-drop functionality.
- Visually Distinct Nodes: Visually distinct node types for sources, transformations, and destinations with color-coded status indicators.
- Pre-defined Transformations Catalogue: Catalogue of pre-defined data transformations (e.g., cleaning, joins, aggregations).
- Real-time Collaboration: Real-time collaboration features including shared pipeline editing.
- Adaptive UI Views: Adaptive UI providing simplified views for data consumers and comprehensive views for engineers.
- Pipeline Export/Import: Export and import pipelines as JSON/YAML files.
- Exhaustive Node Configuration: Comprehensive node configuration panels that are expandable but have sensible defaults. Data officers can see aggregated statistics, while engineers can modify the configurations.
- Data Quality Overlays: Visual cues (badges, color-coding) to represent data quality metrics directly on the canvas.
- Lineage Versioning: Versioning feature for lineages, representing current production status and evolutions. or only the evolutions if it's a new lineage.

## Style Guidelines:

- Primary color: Deep blue (#003366) to represent the robust nature of data flow.
- Background color: Light blue (#E0F2F7) for a clean and professional interface.
- Accent color: Bright cyan (#00FFFF) for interactive elements and highlights, providing contrast and emphasis.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable interface.
- Use minimalist icons to represent different data sources, transformations, and destinations.
- Maintain a consistent and logical layout with clear visual hierarchy for data pipelines.
- Subtle animations to indicate data flow and changes in pipeline status.
- Collapsible panels and a clean canvas minimize clutter and maximize screen real estate.
- Clear section headers and labels for easy scanning and comprehension.
